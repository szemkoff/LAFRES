# Incoming Content

Drop files here for processing into the L.A.F.R.E.S. documentation.

## Folder Structure

```
incoming/
├── articles/     # Word docs, PDFs, text files with article content
├── images/       # Photos, diagrams, charts, graphs
└── data/         # CSV, Excel, raw experimental data
```

## Supported Formats

### Articles
- `.docx` - Microsoft Word
- `.pdf` - PDF documents
- `.txt` - Plain text
- `.md` - Markdown (ready to use)

### Images
- `.jpg`, `.jpeg` - Photos
- `.png` - Screenshots, diagrams
- `.svg` - Vector graphics
- `.gif` - Animated images

### Data
- `.csv` - Comma-separated values
- `.xlsx` - Excel spreadsheets
- `.json` - JSON data

## Processing

Once you add files here, let me know and I'll:
1. Convert articles to Markdown documentation pages
2. Move images to `/website/static/img/`
3. Create data visualizations if needed
4. Update cross-references in existing docs

## Notes

- This folder is for staging only
- Files will be moved to proper locations after processing
- Original files can be kept here for reference
